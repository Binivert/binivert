### Troubleshooting

**Images not loading?**
- Ensure repository is public
- Check exact file paths (case-sensitive)
- Clear browser cache
- Wait 5-10 minutes for GitHub CDN

**Stats widgets not working?**
- These use external services that may have downtime
- Alternative: Use shields.io static badges

**Animations not playing?**
- GitHub caches SVGs; may take time to update
- Ensure SVGs are served from raw.githubusercontent.com
